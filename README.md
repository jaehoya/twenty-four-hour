# 🚀 미니 드라이브 (Mini Drive)

**미니 드라이브는 간편한 파일 업로드 및 관리를 위한 웹 기반 개인 드라이브 서비스입니다.**

이 프로젝트는 현재 개발 중이며, 개인 파일을 안전하게 저장하고 관리할 수 있는 기능을 제공하는 것을 목표로 합니다.
